Mouse Controls

A.    Middle Mouse Button

Middle Mouse Button operations for all modes are identical. These are as follows:

1.     Middle Button Drag: Rotate View

2.     Middle Button+ Shift Drag : Pan View

3.     Middle Button+ Ctrl/Cmd Drag : Zoom View (vertical mouse motion) / Turn (horizontal mouse motion)

4.     Mouse Wheel : Zoom, half-speed with Shift, double-speed with Ctrl/Cmd

B.    Right Mouse Button

Right Mouse Button operation for all modes is identical. It simply brings up the mode specific context menu. In Build mode, different context menu items are provided for open bonds or a bond between two elements or when an atom center is highlighted.

C.    Left Mouse Button

Left Button does something according to the mode. The mode specific operations are given below.

I.      Build Mode

1.     Left Click: Deposits an atom or chunk from the clipboard in the 3D workspace or bonds it to an open bond of an already deposited atom.

2.     Left Drag: When an atom center highlighted (by moving mouse cursor over the atom center), Left Drag on this atom moves the complete chunk to the desired position.

3.     Shift + Left Drag: When an atom center is highlighted, Shift + Left Drag moves the highlighted atom. When an open bond is highlighted, this operation can move the open bond. Thus, to create a bond between atom-A and atom-B:

- Highlight open bond of atom-A

- Press Shift key and also hold the left mouse button down

- Start dragging the mouse. Notice that a white rubber band line appears between the cursor position and the open bond on atom-A

- Now highlight the open bond on atom-B and release left mouse button as well as the Shift key. A bond will be formed between these two atoms.

4.     Ctrl + Left Click: Deletes a highlighted atom or bond.

II.    Select Atoms and Select Chunks Mode

1.     Left Click Operations:

-            Single Left Click:  Selects the atom below the cursor if it is Select Atoms mode and selects chunk if it is Select Chunks mode. In order to select a chunk, the cursor must point on an atom center or a bond within that chunk. Left Click on a jig will select that jig in both, Select Atoms and Select Chunks mode. Single left click also deselects everything else (that doesn’t lie below the mouse cursor)

-            Shift + Left Click: It works just like Single Left Click except that it keeps the other already selected objects as selected (even when those are not directly below the mouse cursor).

-            Ctrl + Left Click: It deselects the objects (atoms / chunks or jigs) that lie directly below the cursor. It doesn’t deselect anything else.

2.     Left Drag Operations:

-            Single Left Drag: Selects atoms or chunks that lie inside the selection rectangle or lasso. While doing so, it also deselects everything (i.e. atoms or chunks) that lie outside the selection rectangle. Note that any of the Left Drag operation will NEVER select the jigs inside the selection rectangle.

-            Shift + Left Drag: It works just like Single Left Drag except that it keeps the other already selected objects as selected (even when those lie outside of the selection rectangle or lasso).

-            Ctrl + Left Drag: It deselects the objects (atoms / chunks or jigs) that lie inside the selection rectangle. It doesn’t deselect anything else.

III.  Move and Fuse Chunks Mode

1.     Left Click Operations: All the left click operations in Move and Fuse Chunks mode are identical to the ones in Select Chunks modes.

2.     Left Drag Operations: For Left Drag operations to work in these modes, one or more chunks must be selected.

-            Single Left Drag: Translates the selected chunk(s) in the same plane as the display screen.

-            Shift + Left Drag: Translates the selected chunk along the axis passing through its center of mass (if it is a translation) and/or rotates the chunk(s) around this axis.

-            Ctrl + Left Drag: Rotates the selected chunk freely around its center of mass.

IV.  Cookie Cutter

1.     Left Click - Free Mouse Drag- Left Click: If an auto-shape is selected, left click will anchor the center or corner of the auto shape depending upon the type of autoshape. The free mouse drag will then expand the selection wire frame. Left click again (inside the 3D workspace) will complete the wire frame drawing and the cookie drawing will be created inside the wire frame.

- If no keyboard key is pressed while doing above operations, it will add whatever is drawn inside the wire frame to the existing cookie drawing in the cookie cutter .

 - If Shift key is held down before the above operation begins, it will keep the already existing cookie drawing inside the wire frame but will delete the drawing outside it.

-  If Shift key is held down before the above operation begins, it will delete the existing cookie drawing inside the wire frame but will keep the other drawings intact.

 

2.     Left Drag Operations: It will draw selection rectangles or lasso outlines inside which cookie drawing will be created (or deleted etc depending on which modifier key is pressed)

V.    Extrude Mode

While in this mode, selection of the chunks from the 3D workspace is not permitted. (However, one can select them from the Model Tree )

1.     Single Left Drag: Drags the repeat units (Repeat Units are the extrude copies of the original chunk to be extruded). The cursor must point on any of the atom centers of the repeat units in order to move them.

VI.  Movie Player

       Left mouse controls are not available in this mode.
