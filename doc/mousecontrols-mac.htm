Right Mouse Button (Context Menus)

Right Mouse Button operations for all modes are identical. It simply brings up the mode specific context menu. In Build Mode, different context menus are provided when the cursor is over an open bond or a bond between two atoms or when an atom center is highlighted.

Mac users with a single mouse button can hold down the Control key to emulate the RMB.

Left Mouse Button (Selection)

Left Mouse Button (LMB) operations will change according to the current mode. Mode specific operations are given below.

Create New Bond

Creates a new bond between two atoms (atom-A and atom-B):

1. Highlight the open bond of atom-A

2. Press and hold SHIFT + LMB

3. Start dragging the mouse. A white rubber band line appears between the cursor position and the open bond on atom-A

4. Now highlight the open bond on atom-B and release the LMB. A bond will be formed between these two atoms.

NOTE: Repeating this procedure on open bonds of the same two atoms will form a double bond between them.

CMD + LMB + Drag

Delete

Deletes a highlighted atom or bond.

Translates/rotates the selected chunk(s) along the X axis. Movement can be constrained to translation or rotation by holding down either the T key (translation only) or the R key (rotation only) at the same time.

Z + LMB + Drag

SHIFT + LMB + Drag

Move Chunks Along Axis

Translates the selected chunk along the axis passing through its center of mass (if it is a translation) and/or rotates the chunk(s) around this axis.

CMD + LMB + Drag

Spin Chunks

Freely rotates the selected chunk around its center of mass.

Extrude Mode

While in Extrude Mode, selection of chunks or jigs from the 3D workspace is not permitted. You can select chunks and jigs from the Model Tree, however.

Action

Result

Description

LBM + Drag

Move

Moves a Repeat Unit, which is one of the extrude copies of the original chunk to be extruded. The cursor must point on any of the atom centers of the repeat units in order to move them. This is useful for connecting repeat units together in a chain, resulting in an extruded chunk.
